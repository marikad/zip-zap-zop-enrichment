Theme Installation

After you unzip the HOPE package you will find several folders labelled as follows:

  • setup_files/
    • WordPress_content/All content/XML Format/Hope-demo-content.xml
    • Redux_options/JSON format/Hope_redux_options.json
    • Theme_customizer_settings/Hope-charity-theme-export.dat
  • plugins/
    • customizer-export-import.zip
    • codecanyon-242431-visual-composer-page-builder-for-wordpress-wordpress-plugin.zip
    • custom_post_types/premium-paypal-manager.zip
    • custom_post_types/woocommerce.zip
  • Licensing
  • Documentation

As of version 3.0 the setup_files folder and plugins are now optional folders. The files located within each of these folders have been integrated directly into the Hope theme through the TGM plugin activator and the Redux demo importer option.


Theme Installation Procedure

To install the HOPE theme follow these steps

  • Step 1: Select "Appearance" and then "Themes" from the WordPress administration menu
  • Step 2: Select "Add New" (located near the top of the page)
  • Step 3: Select "Upload Theme" (located near the top of the page)
  • Step 4: Select "Browse" and locate the Hope-charity-theme.zip file and proceed with installation
  • Step 5: Activate the theme and you're all done.

Installing and activating the required plugins for the Hope WordPress theme:

Upon activating the Hope theme you should receive a notice indicating which plugins are required and they should be installed and activated before proceeding with the 1-click demo importer. See image below for reference:




Importing the WordPress demo content with the 1-click demo importer feature:

To import the WordPress demo content simply go to Hope Options -> Demo Importer and import the Hope Demo data. See image below for reference:



NOTE: The demo importer does not import any customizer settings. The customizer settings are optional and they can be imported through the theme customizer area under Appearance -> Customizer -> Export/Import. If you do not see the Export/Import tab in the customizer area then you will need to install the Customizer export/import plugin which comes bundled with the HOPE theme and can be installed and activated through the WordPress dashboard upon theme activation. The customizer settings file can be found in the following folder location packaged with the Hope theme: setup_files/Theme_customizer_settings/Hope-charity-theme-export.dat

Homepage Setup

  • Step 1:
    • Select Pages -> Add New from the WordPress adminstration menu
  • Step 2:
    • Enter a title such as "Homepage" and press "Publish"
  • Step 3:
    • Select Settings -> Reading from the WordPress adminstration menu (see visual reference below)
  • Step 4:
    • Under "Front page displays" select the radio button labelled "A static page (select below)" and select the page we created called "Homepage" from the select list menu.
  • Step 5:
    • Press "Save Changes" and your custom Homepage will now be assigned to the index page of your website.

Post / Page Options

Page Options

The following custom page options are built into the Luxor theme:

1. Page Header Image

Allows you to upload and assign a designated header image for the page you are creating. Leaving this field empty will display the global header image (if assigned) or the default header background color instead.

2. Page Header Message

Allows you to display a designated header message for the page you are creating.

3. Page Layout

Allows you to select your desired page layout. You can choose to display a sidebar on the left or right of your page or disable the sidebar entirely.

4. Custom Sidebar

This metabox is located on the right hand side column and it allows you to select which sidebar you would like to display on your page. NOTE: this option only applies if your Page Layout option is set to either "Left Sidebar" or "Right Sidebar".

5. Disable Boostrap container for full width content?

Setting this to "Yes" will disable the default bootstrap container around your content allowing you to create full width layouts.

6. Boostrap Container Padding Amount

This setting allows you to control the amount of vertical padding above and beneath your content.

Post Options

The following custom post options are built into the Luxor theme:

1. Post Header Image

Allows you to upload and assign a designated header image for the post you are creating.

2. FontAwesome 4 Icon

Apply a custom FontAwesome 4 icon value for the news post.

3. Tooltip Messsage

Apply a custom tooltip message to the post icon upon rollover.

4. Slides

Upload and display a post slider on the single post page.

5. Page Layout

Allows you to select your desired page layout. You can choose to display a sidebar on the left or right of your page or disable the sidebar entirely.

6. Custom Sidebar

This metabox is located on the right hand side column and it allows you to select which sidebar you would like to display on your page. NOTE: this option only applies if your Page Layout option is set to either "Left Sidebar" or "Right Sidebar".

Custom Post Types

Events Post Type

The Events post type allows you to create and manage events.

To setup an Events page simply create a new "Page" and assign it the "Events Template" file from the Page Attributes metabox. Alternatively you can also use the Event Posts shortcode bundled with Visual Composer.

The Events post type is built with the following custom post options:

1. Event Date

Set the date for your event.

2. Tooltip Message

Enter a custom tooltip message for the icon upon rollover.

3. FontAwesome icon

Enter a custom FontAwesome 4 icon value for the event post.

4. Countdown

Enter a specified date to activate the countdown feature on the event single post page.

5. Panel Header Info

Choose between displaying the event post title or event post meta information.

Gallery Post Type

The Gallery post type allows you to create and manage gallery posts.

To display your gallery posts simply create a new "Page" and assign it the "Gallery Template" file from the Page Attributes metabox. Alternatively you can also use the Gallery Posts shortcode bundled with Visual Composer.

The Gallery post type is built with the following custom post options:

1. Gallery Image

Assign and upload a gallery image.

2. Youtube Video

Assign a Youtube video link.

3. Display Youtube Video

Setting this option to "YES" will play the Youtube video in the pretty photo carousel rather than display the gallery image.

4. Caption

Add a caption to your gallery post. This appears in the expanded view.

Organizers Post Type

The Organizers post type allows you to create and manage organizer profile.

To setup the Organizers page simply create a new "Page" and assign it the "Properties Template" file from the Page Attributes metabox. Alternatively you can also use the Orgainzer Posts shortcode bundled with Visual Composer.

The Organizers post type is built with the following custom post options:

1. Organizer Title

Assign an official title to your organizer profile.

2. Tooltip Message

Enter a custom tooltip message for the icon rollover.

3. Contact Number

Enter a contact number for your organizer profile.

4. Email Address

Enter an email address number for your organizer profile.

5. Personal Website

Enter a web address URL for your organizer profile.

6. Linkedin Profile

Enter a Linkedin URL for your organizer profile.

7. Twitter Profile

Enter a Twitter URL for your organizer profile.

8. Facebook Profile

Enter a Facebook URL for your organizer profile.

9. Google Plus Profile

Enter a Google Plus URL for your organizer profile.

10. Skype Address

Enter a Skype username for your organizer profile.

Shortcodes

NOTE: All of the Hope shortcodes have now been integrated directly into Visual Composer as of version 3.0

We've also updated all of the WordPress demo files with the new VC shortcodes to make each page easier to edit.

All shortcodes can be accessed through the Visual Composer Add Element interface - see image below for reference:





The following is a breakdown of each shortcode accessed from the TinyMCE editor or Visual Composer elements window.

Bootstrap 2 Container

Use the following column shortcodes to properly format your page layout

The columnContainer shortcode is a wrapper for the column shortcode and accepts the following parameters:

  • fullscreen (on or off)
  • bgcolor (any hex value or string value ex. #ffffff or transparent)
  • border (yes or no)
    	
[columnContainer fullscreen="off" bgcolor="#FFFFFF" bgimage="" border="yes"]
Bootstrap Column(s) go here
[/columnContainer]

The columnContainer shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Bootstrap 2 Column

The column shortcode is where you insert the content you wish to display and it accepts the following parameters:

  • span (span1 through span12)
  • alignment (left, center or right)
    	
[column span="span12" alignment=""]

Content goes here

[/column]

The column shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Featured Panel Container

The featuredPanel shortcode is a fullscreen container and acts as wrapper for the bootstrap columnContainer and column shortcodes. The featured panel has a border applied across the top and bottom of the container making it stand out. It accepts the following parameters:

  • bgcolor (apply a hex value for the background color. Ex. #383838)
  • bgimage (apply a background image by entering an absolute path to your image)
  • border_color (apply a hex value for the border color. Ex. #383838)
    	
[featuredPanel bgcolor="" bgimage="" border_color="#3C3C3C"]

Bootstrap container and column(s) go here

[/featuredPanel]

The featuredPanel shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Alert

The alert shortcode displays an alert box and it accepts the following parameters:

  • close (true or false)
  • type (success, danger, info, error)
    	
[alert close="true" type="success"]

Message goes here

[/alert]

The alert shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Hope Button

The hopeButton shortcode (previously button) displays a custom button developed specifically for the HOPE theme and it accepts the following parameters:

  • color (accepts a hex value. Ex. #383838)
  • textcolor (accepts a hex value. Ex. #383838)
  • type (small, medium, large)
  • url (accepts a path to a page or an external absolute path)
  • target (_self or _blank)
  • icon (accpets any FontAwsome 3.2 icon. Ex. icon-angle-right)
    	
[hopeButton color="" textcolor="" type="" url="" target="_self" icon=""]

Button text goes here

[/hopeButton]

The hopeButton shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Progress Bar

The progressBar shortcode displays a progress bar and it accepts the following parameters:

  • percentage (enter a numeric value along with a percent symbol. Ex. 75%)
  • type (success, info, warning, danger)
  • animation (active, static)
    	
[progressBar percentage="50%" type="success" animation="active" /]

The progressBar shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Divider

The divider shortcode displays a thin horizontal bar which can be used to seperate content and it accepts the following parameters:

  • height (set the height of the divider)
  • bg_color (apply a color to the divider - accepts a hex value. Ex. #383838)
  • margin (a postitive numeric value)
    	
[divider height="1" bg_color="orange" margin="20" /]

The divider shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Single Post

The singlePost shortcode displays a single post and it accepts the following parameters:

  • id (enter the id number of your post)
    	
[singlePost id="1" /]

The singlePost shortcode can be easily inserted into the visual editor using the TinyMCE editor.

YouTube Video

The youtubeVideo shortcode displays a YouTube video and it accepts the following parameters:

  • id (accepts a YouTube video id. Ex. bPcQDI-GDKk)
  • width (set the width of your video - this only applies if responsive is set to 0)
  • height (set the height of your video)
  • responsive (enter 0 for fixed width or 1 for full width responsiveness)
    	
[youtubeVideo id="bPcQDI-GDKk" width="300" height="250" responsive="0" /]

The youtubeVideo shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Vimeo Video

The vimeoVideo shortcode displays a Vimeo video and it accepts the following parameters:

  • id (accepts a Viemo video id. Ex. 88896418)
  • width (set the width of your video - this only applies if responsive is set to 0)
  • height (set the height of your video)
  • responsive (enter 0 for fixed width or 1 for full width responsiveness)
    	
[vimeoVideo id="88896418" width="300" height="250" responsive="0" /]

The vimeoVideo shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Tab System (tabGroup and tabItem)

The tab system is a combination of two shortcodes working in unison - the tabGroup and the tabItem. The tabGroup is a wrapper for the tabItem and there can be multiple tabItem shortcodes within the tabGroup shortcode. Each tabItem creates a new a tab along with its content area.

The tabGroup shortcode does not take any parameters but the tabItem takes the following parameter:

  • title (insert a title for the tab)
    	
[tabGroup]
[tabItem title="Tab 1"]Tab 1 content goes here[/tabItem]
[tabItem title="Tab 2"]Tab 2 content goes here[/tabItem]
[/tabGroup]

The tab system shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Accordion System (accordionGroup and accordionItem)

The accordion system is a combination of two shortcodes working in unison - the accordionGroup and the accordionItem. The accordionGroup is a wrapper for the accordionItem and there can be multiple accordionItem shortcodes within the accordionGroup shortcode. Each accordionItem creates a new a accordion along with its content area.

The accordionGroup shortcode does not take any parameters but the accordionItem takes the following parameters:

  • title (insert a title for the accordion)
  • icon (insert any FontAwesome 3 icon. Ex. icon-angle-right)
    	
[accordionGroup]
[accordionItem title="Accordion 1"]Accordion 1 content goes here[/accordionItem]
[accordionItem title="Accordion 2"]Accordion 2 content goes here[/accordionItem]
[/accordionGroup]

The accordion system shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Panel Header

The panelHeader shortcode is used to display a stylish header designed specifically for the HOPE theme and it takes the following parameters:

  • icon (insert a FontAwesome 3 icon. Ex. icon-angle-right)
  • link (insert a link to a page or an absolute path to an external page)
  • target (_self or _blank)
  • tip (enter a message to activate the tooltip or leave blank to omit the tooltip)
  • bgcolor (apply a hex value to change the color of the panel header. Ex. #383838)
  • marginbottom (enter a positive numeric value to add spacing along the botto of the panel header)
    	
[panelHeader icon="icon-file" link="" target="_self" tip="" bgcolor="orange" marginbottom="10"]
Panel Header text goes here
[/panelHeader]

The panelHeader shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Image Panel

The imagePanel shortcode can be used to add an interactive image and it takes the following parameters:

  • tip (enter a message to activate the tooltip or leave blank to omit the tooltip)
  • icon (insert a FontAwesome 3 icon. Ex. icon-angle-right)
  • title (enter a descriptive title)
  • link (insert a link to a page or an absolute path to an external page)
  • image (enter an absolute path to your desired image)
    	
[imagePanel tip="" icon="" title="" link="" image="" /]

The imagePanel shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Google Map

The googleMap shortcode can be used to add an interactive map to your page and it takes the following parameters:

  • id (enter a unique id name for your map)
  • zoom (enter a value between 1 and 20 to set the initial zoom of the map)
  • latitude (enter the latitude of your location. Ex.43.656885)
  • longitude (enter the longitude of your location. Ex.-79.383904)
  • message (enter a message for the marker box)
  • responsive (0 for fixed width or 1 for responsiveness)
  • width (set the width of the map - this only applies if the responsive parameter is set to 0)
  • height (set the height of the map)

NOTE: Use the free online tool, iTouchMap, to get the latitude and longitude for your map.

    	
[googleMap id="anotherMap" zoom="13" latitude="43.656885"  longitude="-79.383904" message="We are here" responsive="1" width="300"   height="300" /]

The googleMap shortcode can be easily inserted into the visual editor using the TinyMCE editor.

Language Support

The HOPE theme can be translated to your language of choice by editing the localization.pot catalogue file located in the theme's languages folder. This file can be opened and edited with the POEdit software - you can download POEdit here. Once you have inserted your translations and assigned your language to the file you can go ahead and save the file - a .mo and .po file will be automatically generated upon save.

Please make sure to save your file with the correct language code that you require. So for example: if you are creating a translation file for french (France) then the file should be saved as fr_FR.po. You can find a full list supported translation codes here

To switch the language of your WordPress website go to Settings -> General and scroll down to the Site Language list and select your desired language.

If you require multiple languages then you will have to use a plugin such as WPML - WordPress itself does not support multiple languages.

Helper Classes

The following is a list of helper classes that be applied to shortcodes:

  • pm-center (can be applied to any bootstrap shortcode. This will center all the content within your bootstrap columns)
  • pm-center-mobile (can be applied to any bootstrap shortcode. This will center all the content within your bootstrap columns on mobile devices only)
  • pm-page-scroll (can be assigned to the standardButton shortcode to trigger a page scroll effect)
  • pm-column-spacing (adds a 30 pixel spacing to bootstrap columns)
  • pm-columnPadding10 up to pm-columnPadding120 (each class jumps up by an increment of 10). You can use these classes to add more vertical spacing on bootstrap rows/columns or Visual composer elements if required.
  • pm-no-margin (useful if you wish to remove all margins on an element)
  • pm-no-padding (useful if you wish to remove all padding on an element)
  • pm-clear-element (if you are experiencing float issues then you can apply this class to your element(s) to clear the floats)
  • pm-show-overflow (can be applied to any element or Visual Composer row to display overflow content)
